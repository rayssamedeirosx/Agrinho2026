# Agrinho2026

COLHEITA DO FUTURO!

OBJETIVO: O site ''Colheita do Futuro'' foi desenvolvido com o principal objetivo de mostrar que o meio agrícola pode ser forte e produzir muito, mas ainda assim ser sustentável, preservando sempre o bem-estar do meio ambiente.

JUSTIFICATIVA: Após observar que muitas pessoas pensam que ter um agronegócio forte não pode ser sinônimo de preservar a natureza, surgiu a ideia da ''Colheita do Futuro'' para mostrar que é perfeitamente possível manter um agro forte sem prejudicar o meio ambiente. Para provar isso na prática, o aplicativo funciona como uma plataforma inteligente que ajuda o produtor rural através de uma Consulta Inteligente, que ensina o descarte correto de resíduos do campo (como óleos, pilhas, pneus e defensivos), e de uma Calculadora Ambiental, que mede na hora os milhões de litros de água salvos e os anos de poluição evitados. O sistema ainda traz Gráficos Estatísticos e indica os Postos de Coleta Oficiais do Paraná para que a reciclagem aconteça de verdade.
